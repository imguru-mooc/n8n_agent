n8n 로컬 호스트 설정 

node js download : https://nodejs.org/ko/download

node js 설치 확인
C:\> node
node js 버전 확인
C:\> node -v
node package manager 설치 확인
C:\> npm

n8n 설치
C:\> npm install -g n8n

n8n 서버 기동
C:\> n8n start

크롬에서 
http://localhost:5678 실행
