## 실습 7: 설문 & 피드백 자동 수집 시스템

### 목표
수업 종료 후 설문 이메일을 자동 발송하고, 응답을 수집하여 만족도를 자동 집계합니다.

---

### Step 1: 사전 준비 — 피드백 DB 시트 만들기

1. Google Sheets에서 새 스프레드시트 생성: **"교육피드백_DB"**

2. **Sheet1** 이름을 **"피드백"** 으로 변경 (시트 탭 더블클릭)

3. 1행(헤더) 입력:

   | A | B | C | D | E | F |
   |---|---|---|---|---|---|
   | 이름 | 과정명 | 만족도 | 피드백 | 수업날짜 | 응답일시 |

4. 테스트용 샘플 데이터 5건 입력:

   | 이름 | 과정명 | 만족도 | 피드백 | 수업날짜 | 응답일시 |
   |------|--------|--------|--------|----------|----------|
   | 홍길동 | AI입문 | 5 | 실습이 많아서 좋았습니다. 특히 n8n 활용 부분이 유익했어요. | 2026-03-20 | 2026-03-20 17:30 |
   | 김영희 | AI입문 | 4 | 전반적으로 좋았으나 속도가 좀 빨랐습니다. 복습 시간이 더 있으면 좋겠어요. | 2026-03-20 | 2026-03-20 17:45 |
   | 박철수 | AI입문 | 3 | 이론 부분이 좀 어려웠습니다. 초보자를 위한 설명이 더 필요합니다. | 2026-03-20 | 2026-03-20 18:00 |
   | 이지은 | AI입문 | 5 | 강사님 설명이 명확하고 실습 예제가 실무적이어서 매우 만족합니다. | 2026-03-20 | 2026-03-20 18:15 |
   | 최민수 | AI입문 | 4 | 좋은 교육이었습니다. 다만 교재가 있으면 더 좋겠습니다. | 2026-03-20 | 2026-03-20 18:30 |

5. 새 시트 탭 추가 (하단 **+** 클릭) → 이름을 **"리포트"** 로 변경

6. 리포트 시트 헤더:

   | A | B | C | D | E |
   |---|---|---|---|---|
   | 과정명 | 평균만족도 | 응답수 | 집계일시 | AI요약 |

### Step 2: 설문 발송 워크플로우

1. 새 워크플로우: **"실습7_피드백자동수집"**

2. **Manual Trigger** 추가 (테스트용 — 실제로는 Schedule Trigger)

3. **Google Sheets** (Read Rows) → "수강생_명단" 시트에서 수강생 읽기

4. **Gmail** 노드 추가:
   - **To**: `{{ $json.이메일 }}`
   - **Subject**: `[{{ $json.과정 }}] 수업은 어떠셨나요? 피드백을 남겨주세요 📝`
   - **Message**:
     ```html
     <h2>{{ $json.이름 }}님, 오늘 수업은 어떠셨나요?</h2>
     <p>{{ $json.과정 }} 수업에 참여해주셔서 감사합니다.</p>
     <p>더 좋은 교육을 위해 간단한 피드백을 부탁드립니다.</p>
     <p><a href="https://forms.google.com/your-form-link" 
           style="background:#0D7377;color:white;padding:12px 24px;
                  text-decoration:none;border-radius:6px;font-weight:bold;">
        📝 설문 참여하기
     </a></p>
     <p>소요 시간: 약 2분</p>
     ```

5. **"Test workflow"** 실행하여 이메일 발송 확인

### Step 3: 만족도 자동 집계 워크플로우

1. 같은 워크플로우에 새 흐름 추가 (또는 별도 워크플로우)

2. **Manual Trigger** (두 번째)

3. **Google Sheets** (Read Rows) → "교육피드백_DB" → "피드백" 시트 전체 읽기

4. **Code** 노드 추가:
   - **Language**: JavaScript

5. Code 노드에 아래 코드 입력:

   ```javascript
   // 모든 피드백 데이터 가져오기
   const items = $input.all();
   
   // 만족도 점수 추출 (숫자로 변환)
   const scores = items.map(item => Number(item.json.만족도));
   
   // 평균 계산
   const sum = scores.reduce((a, b) => a + b, 0);
   const avg = sum / scores.length;
   
   // 과정명 (첫 번째 항목에서 가져옴)
   const courseName = items[0].json.과정명;
   
   // 결과 반환
   return [{
     json: {
       과정명: courseName,
       평균만족도: avg.toFixed(1),
       응답수: scores.length,
       최고점: Math.max(...scores),
       최저점: Math.min(...scores),
       집계일시: new Date().toISOString().slice(0, 19).replace('T', ' ')
     }
   }];
   ```

6. **"Test step"** 버튼으로 Code 노드만 테스트:

   ```json
   {
     "과정명": "AI입문",
     "평균만족도": "4.2",
     "응답수": 5,
     "최고점": 5,
     "최저점": 3,
     "집계일시": "2026-03-22 15:00:00"
   }
   ```

> ✅ **확인**: 평균만족도, 응답수가 정확히 계산되면 성공!

---

### Step 4: 만족도 저하 시 알림

1. Code 노드 뒤에 **IF** 노드 추가:
   - Value 1: `{{ Number($json.평균만족도) }}`  (Expression 모드)
   - Operation: `is less than`
   - Value 2: `4.0`

2. IF **TRUE** (만족도 4.0 미만) → **Gmail** 노드:
   - **To**: `admin@example.com` (관리자 이메일)
   - **Subject**: `⚠️ [{{ $json.과정명 }}] 만족도 경고 — 평균 {{ $json.평균만족도 }}점`
   - **Message**:
     ```html
     <h2>⚠️ 교육 만족도 경고</h2>
     <table border="1" cellpadding="8" style="border-collapse:collapse;">
       <tr><td><strong>과정명</strong></td><td>{{ $json.과정명 }}</td></tr>
       <tr><td><strong>평균 만족도</strong></td><td style="color:red;font-weight:bold;">{{ $json.평균만족도 }} / 5.0</td></tr>
       <tr><td><strong>응답 수</strong></td><td>{{ $json.응답수 }}명</td></tr>
       <tr><td><strong>집계 일시</strong></td><td>{{ $json.집계일시 }}</td></tr>
     </table>
     <p>만족도가 기준(4.0) 이하입니다. 원인 분석 및 개선 조치가 필요합니다.</p>
     ```

3. IF **FALSE** (만족도 4.0 이상) → **Google Sheets** (Append Row):
   - Document: "교육피드백_DB"
   - Sheet: "리포트"
   - 집계 결과 저장

## 실습 8: 수업 자료 자동 배포 시스템

### 목표
수업 전날 해당 과정 자료를 Google Drive에서 찾아 수강생에게 자동 발송합니다.

---

### Step 1: 사전 준비 — Google Drive 폴더 구성

1. Google Drive에 아래 폴더 구조를 만듭니다:

   ```
   📁 교육자료
     ├── 📁 AI입문
     │   ├── 📄 AI입문_1주차_강의자료.pdf
     │   └── 📄 AI입문_실습가이드.pdf
     └── 📁 데이터분석
         ├── 📄 데이터분석_1주차_강의자료.pdf
         └── 📄 데이터분석_실습가이드.pdf
   ```

2. 각 폴더에 테스트용 파일을 1~2개 업로드합니다 (아무 PDF나 가능)

---

### Step 2: Google Drive Credential 확인

- 2교시에서 만든 "My Google" Credential이 Google Drive 접근 권한도 포함하고 있으면 그대로 사용
- 없으면 Credentials에서 **"Google Drive OAuth2 API"** 추가 등록

---

### Step 3: 워크플로우 구성

1. 새 워크플로우: **"실습8_자료자동배포"**

2. **Manual Trigger** 추가

3. **Google Sheets** (Read Rows) → "수업일정" 시트 읽기

4. **IF** 노드: 내일 수업인지 필터링
   - `{{ $json.수업날짜 }}` is equal to `{{ $now.plus(1, 'day').format('YYYY-MM-DD') }}`

5. IF TRUE → **Google Drive** 노드:
   - **Operation**: `Search Files`
   - **Query**: `'{{ $json.과정명 }}' in parents`
     > 또는 폴더 ID를 직접 지정

6. Google Drive 뒤에 **Google Sheets** (Read) → "수강생_명단" 에서 해당 과정 수강생 조회

7. → **Gmail** 노드:
   - **To**: `{{ $json.이메일 }}`
   - **Subject**: `[{{ $json.과정명 }}] 내일 수업 자료를 보내드립니다 📚`
   - **Message**:
     ```html
     <h2>{{ $json.이름 }}님, 내일 수업 자료입니다!</h2>
     <p>내일 <strong>{{ $json.과정명 }}</strong> 수업에서 사용할 자료를 보내드립니다.</p>
     <p>📎 아래 링크에서 다운로드해주세요:</p>
     <ul>
       <li><a href="{{ $json.webViewLink }}">{{ $json.name }}</a></li>
     </ul>
     <p>수업 전에 미리 확인해주시면 좋습니다. 감사합니다!</p>
     ```

8. 마지막에 **Google Sheets** (Append Row) → 발송 로그 기록:
   - Document: 별도 "발송로그" 시트
   - 필드: 이름, 이메일, 과정명, 파일명, 발송일시

---

### Step 4: 테스트

1. "수업일정" 시트에 내일 날짜의 수업 데이터가 있는지 확인
2. Manual Trigger로 테스트 실행
3. 이메일 발송 및 발송 로그 기록 확인

> ✅ **산출물**: 수업 자료 자동 배포 워크플로우
